# 📋 Инструкция по деплою бэкэнда на TimeWeb

## ✅ Что уже готово

1. ✅ **Файл `backend/main_timeweb.py`** - готовый сервер FastAPI
2. ✅ **Файл `backend/requirements_timeweb.txt`** - зависимости
3. ✅ **Файл `backend/yandex_disk_api.py`** - функции для работы с Яндекс Диском
4. ✅ **Файл `src/lib/yandexDiskApiTimeweb.ts`** - API клиент для фронтенда

## 🚀 Шаги деплоя

### Шаг 1: Загрузить файлы на TimeWeb

**Вариант А: Через Git (рекомендуется)**
1. Создайте репозиторий на GitHub (если еще нет)
2. В панели TimeWeb → Сайты → Добавить сайт
3. Выберите "Python приложение"
4. Укажите репозиторий Git
5. Укажите путь: `Helper2/backend`
6. Команда запуска: `uvicorn main_timeweb:app --host 0.0.0.0 --port 8000`

**Вариант Б: Через FTP**
1. Подключитесь к FTP TimeWeb
2. Загрузите файлы:
   - `main_timeweb.py`
   - `yandex_disk_api.py`
   - `requirements_timeweb.txt`
   - `.env` (создайте вручную)

### Шаг 2: Настроить переменные окружения

В панели TimeWeb → Переменные окружения добавьте:

```env
YANDEX_DISK_PUBLIC_KEY=BSNe4agC5hSAoA
PORT=8000
```

**Или создайте файл `.env` в корне проекта:**
```env
YANDEX_DISK_PUBLIC_KEY=BSNe4agC5hSAoA
PORT=8000
```

### Шаг 3: Установить зависимости

В панели TimeWeb → Python → Установить зависимости:
```bash
pip install -r requirements_timeweb.txt
```

### Шаг 4: Запустить сервер

В панели TimeWeb → Python → Запустить приложение:
```bash
uvicorn main_timeweb:app --host 0.0.0.0 --port 8000
```

### Шаг 5: Обновить фронтенд

После деплоя получите URL вашего бэкэнда (например: `https://ваш-домен.timeweb.ru`)

**Обновите `src/lib/yandexDiskApi.ts`:**

Замените:
```typescript
const EDGE_FUNCTION_URL = `${SUPABASE_URL}/functions/v1/yandex-disk`;
```

На:
```typescript
const BACKEND_API_URL = 'https://ваш-домен.timeweb.ru/api/yandex-disk';
```

И обновите функции:

```typescript
export const getYandexDiskFiles = async (folderPath?: string): Promise<YandexDiskFilesResponse> => {
  const url = `${BACKEND_API_URL}/files${folderPath ? `?folder_path=${encodeURIComponent(folderPath)}` : ''}`;
  
  const response = await fetch(url);
  if (!response.ok) {
    throw new Error(`Ошибка ${response.status}`);
  }
  return await response.json();
};
```

**Или просто скопируйте содержимое `yandexDiskApiTimeweb.ts` в `yandexDiskApi.ts`** и замените `BACKEND_API_URL` на ваш домен.

## ✅ Проверка работы

1. Откройте: `https://ваш-домен.timeweb.ru/`
   - Должно вернуться: `{"status": "ok", ...}`

2. Проверьте API: `https://ваш-домен.timeweb.ru/api/yandex-disk/files`
   - Должен вернуться список файлов

3. Откройте фронтенд и перейдите в раздел "Яндекс Диск"
   - Должны отображаться файлы и папки

## 🔧 Решение проблем

### Ошибка: "Module not found"
- Убедитесь, что зависимости установлены: `pip install -r requirements_timeweb.txt`

### Ошибка: "YANDEX_DISK_PUBLIC_KEY не настроен"
- Проверьте переменные окружения в панели TimeWeb
- Или создайте файл `.env` с нужными переменными

### Ошибка: CORS
- Убедитесь, что в `main_timeweb.py` указаны правильные `allow_origins`
- В продакшене укажите конкретный домен фронтенда вместо `"*"`

### Порт не работает
- Проверьте, что порт указан правильно в настройках TimeWeb
- По умолчанию используется порт 8000

## 📝 Структура API

### GET `/api/yandex-disk/files?folder_path=...`
Получить список файлов из папки

### GET `/api/yandex-disk/download-link?file_path=...`
Получить ссылку для скачивания файла

### GET `/api/yandex-disk/view-link?file_path=...`
Получить ссылку для просмотра файла
