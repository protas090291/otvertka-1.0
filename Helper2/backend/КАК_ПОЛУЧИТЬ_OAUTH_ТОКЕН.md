# Как получить OAuth токен для Яндекс Диска

## Для доступа к корню Яндекс Диска нужен OAuth токен

Чтобы видеть корень Яндекс Диска с папками:
- "Вишневый_сад-3_Приемка работ ФОТО"
- "Вишневый_сад-3_Рабочая"
- "Вишневый_сад-3_для_Заказчика"
- "Скриншоты"

Нужен OAuth токен, а не публичный ключ.

## Шаги получения OAuth токена

### 1. Создать приложение в Яндекс OAuth

1. Перейдите на https://oauth.yandex.ru/
2. Нажмите "Создать новое приложение"
3. Заполните форму:
   - **Название:** "Otvertka Demo" (или любое другое)
   - **Платформы:** Выберите "Веб-сервисы"
   - **Callback URI:** `http://localhost:8000/oauth/callback` (для разработки)
   - **Права доступа:** Выберите "cloud_api:disk.read" (чтение файлов)
4. Сохраните приложение

### 2. Получить Client ID и Client Secret

После создания приложения вы получите:
- **Client ID** (идентификатор приложения)
- **Client Secret** (секретный ключ)

### 3. Получить токен через браузер

Откройте в браузере:
```
https://oauth.yandex.ru/authorize?response_type=token&client_id=ВАШ_CLIENT_ID
```

Замените `ВАШ_CLIENT_ID` на ваш Client ID.

После авторизации в URL будет токен вида:
```
https://oauth.yandex.ru/verification_code?code=КОД
```

Или если используется `response_type=token`:
```
http://localhost:8000/oauth/callback#access_token=ВАШ_ТОКЕН&token_type=bearer&expires_in=31536000
```

### 4. Добавить токен в .env

Создайте или обновите файл `backend/.env`:

```env
YANDEX_DISK_TOKEN=ВАШ_ТОКЕН
# Закомментируйте или удалите публичный ключ для доступа к корню
# YANDEX_DISK_PUBLIC_KEY=BSNe4agC5hSAoA
```

### 5. Перезапустить бэкэнд

```bash
cd Helper2/backend
python3 main_timeweb.py
```

## Альтернатива: Использовать существующий скрипт

В проекте есть файл `backend/exchange_code_for_token.py` - используйте его для получения токена.

## Проверка

После добавления токена проверьте:
```bash
curl http://localhost:8000/api/yandex-disk/files
```

Должны вернуться 4 папки из корня Яндекс Диска.
