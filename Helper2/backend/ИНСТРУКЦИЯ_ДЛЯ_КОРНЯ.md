# Инструкция: Как открыть корень Яндекс Диска

## Проблема

Сейчас используется публичный ключ `BSNe4agC5hSAoA`, который открывает конкретную папку с 51 папкой квартир.

Но вам нужно видеть **корень Яндекс Диска** с 4 папками:
1. "Вишневый_сад-3_Приемка работ ФОТО"
2. "Вишневый_сад-3_Рабочая"
3. "Вишневый_сад-3_для_Заказчика"
4. "Скриншоты"

## Решение: Использовать OAuth токен

Для доступа к корню Яндекс Диска нужен **OAuth токен**, а не публичный ключ.

### Вариант 1: Если у вас уже есть OAuth токен

1. Откройте файл `backend/.env`
2. Добавьте или обновите:
   ```env
   YANDEX_DISK_TOKEN=ваш_oauth_токен
   ```
3. Закомментируйте или удалите публичный ключ:
   ```env
   # YANDEX_DISK_PUBLIC_KEY=BSNe4agC5hSAoA
   ```
4. Перезапустите бэкэнд:
   ```bash
   cd Helper2/backend
   python3 main_timeweb.py
   ```

### Вариант 2: Получить новый OAuth токен

1. **Создайте приложение на https://oauth.yandex.ru/**
   - Название: "Otvertka Demo"
   - Платформы: "Веб-сервисы"
   - Права: "cloud_api:disk.read"

2. **Получите Client ID** из настроек приложения

3. **Откройте в браузере:**
   ```
   https://oauth.yandex.ru/authorize?response_type=token&client_id=ВАШ_CLIENT_ID
   ```

4. **После авторизации** скопируйте токен из URL (параметр `access_token=...`)

5. **Добавьте токен в `backend/.env`:**
   ```env
   YANDEX_DISK_TOKEN=скопированный_токен
   ```

6. **Перезапустите бэкэнд**

### Вариант 3: Использовать существующий скрипт

В проекте есть `backend/exchange_code_for_token.py` - используйте его для получения токена.

## Проверка

После добавления токена проверьте:
```bash
curl http://localhost:8000/api/yandex-disk/files
```

Должны вернуться 4 папки из корня Яндекс Диска.

## Важно

- **OAuth токен** дает доступ к **корню** Яндекс Диска (все ваши папки)
- **Публичный ключ** дает доступ только к **конкретной публичной папке**

Для вашей задачи нужен **OAuth токен**.
