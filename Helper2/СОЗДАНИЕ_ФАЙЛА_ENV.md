# ⚙️ Создание файла .env для Яндекс Диска

## 📝 Инструкция по созданию файла .env

### Шаг 1: Создайте файл

1. **Откройте папку:** `Helper2/backend/`
2. **Создайте новый файл** с именем `.env` (без расширения)
3. **Важно:** Файл должен называться именно `.env` (с точкой в начале)

### Шаг 2: Скопируйте и вставьте этот текст

```env
# ============================================
# Настройки Яндекс Диска
# ============================================

# OAuth токен Яндекс Диска (пример — замените на свой реальный токен)
YANDEX_DISK_TOKEN=ВАШ_ТОКЕН_ЯНДЕКС_ДИСКА_ЗДЕСЬ

# Путь к папке на Яндекс Диске
# УКАЖИТЕ ПУТЬ К ПАПКЕ ЗДЕСЬ (см. инструкцию ниже)
YANDEX_DISK_FOLDER_PATH=/
```

### Шаг 3: Укажите путь к папке

**Замените `/` на путь к вашей папке.**

**Как найти путь:**
1. Откройте Яндекс Диск: https://disk.yandex.ru/
2. Перейдите в нужную папку
3. Посмотрите путь в адресной строке
4. Возьмите часть после `/client/disk/` и добавьте `/` в начало

**Примеры:**
- Если папка называется "Документы": `/Документы`
- Если папка в подпапке: `/Работа/Строительство/Документы`
- Если папка в общем доступе: `/Общий доступ/Документы проекта`

**Итоговый файл должен выглядеть так:**

```env
YANDEX_DISK_TOKEN=ВАШ_ТОКЕН_ЯНДЕКС_ДИСКА_ЗДЕСЬ
YANDEX_DISK_FOLDER_PATH=/Документы проекта
```

### Шаг 4: Сохраните файл

Сохраните файл `.env` в папке `Helper2/backend/`

### Шаг 5: Перезапустите backend сервер

После создания файла `.env`:
1. Остановите backend сервер (если запущен)
2. Запустите его заново
3. Токен будет загружен автоматически

## ✅ Проверка

После настройки:

1. Откройте приложение в браузере
2. Перейдите в раздел "Документы из Яндекс Диска"
3. Файлы должны загрузиться автоматически!
