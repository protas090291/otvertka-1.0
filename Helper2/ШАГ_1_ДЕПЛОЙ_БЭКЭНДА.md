## 📝 Пошаговая инструкция

### Шаг 1.1: Откройте панель TimeWeb Cloud

1. Откройте браузер
2. Перейдите на сайт TimeWeb Cloud: https://timeweb.cloud
3. Войдите в свой аккаунт

---

### Шаг 1.2: Создайте новое приложение

1. В левом меню найдите раздел **"Сайты"** или **"Приложения"**
2. Нажмите кнопку **"Добавить сайт"** или **"Создать приложение"**
3. Выберите тип: **"Python приложение"** или **"Python"**

---

### Шаг 1.3: Подключите GitHub репозиторий

1. Найдите раздел **"Подключение репозитория"** или **"Git"**
2. Выберите **"GitHub"** как источник
3. Если нужно авторизоваться в GitHub - сделайте это
4. Найдите репозиторий: **`protas090291/otvertka-1.0`**
5. Выберите его

---

### Шаг 1.4: Настройте путь к приложению

1. Найдите поле **"Путь к приложению"** или **"Application path"**
2. Введите: **`Helper2/backend`**
   - Это путь к папке с Python файлами внутри репозитория

---

### Шаг 1.5: Настройте команду запуска

1. Найдите поле **"Команда запуска"** или **"Start command"**
2. Введите: **`uvicorn main_timeweb:app --host 0.0.0.0 --port 8000`**

### Шаг 1.9: Запустите приложение

1. Найдите кнопку **"Запустить"** или **"Start"** или **"Deploy"**
2. Нажмите на неё
3. Дождитесь запуска (обычно 30-60 секунд)

---

### Шаг 1.10: Получите URL вашего бэкенда

1. После запуска TimeWeb покажет URL вашего приложения
2. Обычно это что-то вроде: `https://ваш-проект.timeweb.ru` или `https://ваш-проект.timeweb.cloud`
3. **ВАЖНО:** Запишите этот URL! Он понадобится для фронтенда
4. Пример: `https://otvertka-backend.timeweb.ru`

---

### Шаг 1.11: Проверьте работу бэкенда

1. Откройте новый таб в браузере
2. Введите URL вашего бэкенда (например, `https://otvertka-backend.timeweb.ru`)
3. Нажмите Enter
4. Должна открыться страница с текстом:
   ```json
   {
     "status": "ok",
     "service": "Yandex Disk API Proxy",
     "version": "1.0.0"
   }
   ```
5. Если видите этот текст - бэкенд работает! ✅

---

### Шаг 1.12: Привяжите домен (опционально, но рекомендуется)

Если у вас есть домен и вы хотите использовать его для API:

1. В настройках приложения найдите раздел **"Домены"** или **"Domains"**
2. Нажмите **"Добавить домен"** или **"Add domain"**
3. Введите поддомен для API, например: **`api.ваш-домен.ru`**
4. TimeWeb покажет инструкции по настройке DNS
5. Настройте DNS у вашего регистратора домена:
   - Создайте **CNAME запись**: `api` → указанный TimeWeb адрес
   - Или **A запись**: `api` → IP адрес TimeWeb (узнайте в панели)
6. Дождитесь применения DNS (обычно 5-30 минут)

## 🆘 Если что-то не работает

### Проблема: Приложение не запускается

**Решение:**
1. Проверьте логи в панели TimeWeb (раздел "Логи" или "Logs")
2. Убедитесь, что команда запуска правильная: `uvicorn main_timeweb:app --host 0.0.0.0 --port 8000`
3. Проверьте, что путь к приложению правильный: `Helper2/backend`

### Проблема: Ошибка "Module not found"

**Решение:**
1. Убедитесь, что зависимости установлены
2. Проверьте, что файл `requirements_timeweb.txt` существует в репозитории
3. Попробуйте установить зависимости вручную через SSH (если есть доступ)

### Проблема: Ошибка "YANDEX_DISK_TOKEN не установлен"

**Решение:**
1. Проверьте, что переменная окружения `YANDEX_DISK_TOKEN` добавлена
2. Убедитесь, что значение правильное (без пробелов в начале и конце)
3. Перезапустите приложение после добавления переменных
