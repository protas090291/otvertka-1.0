## Как проверить

1. **Проверка бэкэнда:**
   ```bash
   curl http://localhost:8000/
   # Должно вернуться: {"status":"ok","service":"Yandex Disk API Proxy","version":"1.0.0"}
   ```

2. **Проверка API:**
   ```bash
   curl http://localhost:8000/api/yandex-disk/files
   # Должен вернуться список файлов (51 папка)
   ```

3. **Проверка фронтенда:**
   - Откройте приложение: http://localhost:5175 (или ваш порт фронтенда)
   - Перейдите в раздел "Яндекс Диск"
   - Должны отображаться 51 папка

## Остановка бэкэнда

Если нужно остановить бэкэнд:
```bash
# Найти процесс
ps aux | grep "main_timeweb\|uvicorn"

# Остановить (замените PID на номер процесса)
kill <PID>
```

## Перезапуск бэкэнда

```bash
cd Helper2/backend
python3 main_timeweb.py
```

## После деплоя на TimeWeb

После деплоя на TimeWeb обновите `src/lib/yandexDiskApi.ts`:

Замените:
```typescript
const BACKEND_API_URL = 'http://localhost:8000/api/yandex-disk';
```

На:
```typescript
const BACKEND_API_URL = 'https://ваш-домен.timeweb.ru/api/yandex-disk';
```
